## 🏠 1. House Price Prediction

### 📌 Objective
To predict house sale prices based on various property features using regression algorithms.

### 🔧 Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

### 📊 Workflow
- Data Cleaning and Preprocessing
- Exploratory Data Analysis (EDA)
- Feature Selection
- Train-Test Split
- Model Training (Linear Regression, Random Forest)
- Model Evaluation (R² Score, MAE, MSE, RMSE)

### 🎯 Result
Compared multiple regression models and identified the best-performing model for predicting house prices.
